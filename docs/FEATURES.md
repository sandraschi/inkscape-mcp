# Features

## AI SVG Generation

Transform natural language into professional SVG vector graphics using advanced AI models.

- **Conversational Creation**: Describe designs in plain English
- **Style Presets**: geometric, organic, technical, heraldic, abstract
- **Quality Levels**: draft, standard, high, ultra
- **Iterative Refinement**: Improve designs through conversation
- **Post-Processing**: Automatic Inkscape optimization

## Professional Vector Operations

### Generative Tools
- SVG generation from text descriptions
- Barcode and QR code creation
- Mesh gradient generation
- Text-to-path conversion
- Image tracing (raster to vector)

### Geometric Operations
- Boolean operations (union, difference, intersection)
- Path inset/outset for borders and effects
- Shape manipulation and transformation

### Path Engineering
- Path simplification and optimization
- Compound path management
- Object-to-path conversion
- SVG cleaning and compression

### Analysis & Measurement
- Document statistics and analysis
- Object dimension queries
- Path complexity measurement
- Layer and object enumeration

### Specialized Workflows
- DXF export for CAD applications
- Layer-to-files export for texture atlases
- Canvas fitting for clean imports
- VR/Unity pipeline optimization

## Technical Excellence

- **Cross-platform**: Windows, macOS, Linux support
- **FastMCP 2.14.3+**: Modern MCP architecture
- **Async operations**: Non-blocking execution
- **Robust error handling**: Comprehensive validation
- **Security focused**: Input validation and isolation
